## 4. TAXONOMÍA DE ARCHIVOS Y CONOCIMIENTO (Nomenclatura)
Para que el Agente IA comprenda el contexto de cada documento de un vistazo, se utilizan las siguientes etiquetas obligatorias en los títulos (H1):

*   **[SYS] (Sistema):** Define las leyes estructurales, arquitectura base, puertos y protocolos innegociables. Son los cimientos.
*   **[PRJ] (Proyecto):** Define la estructura, objetivos y flujos de un producto específico o desarrollo para un cliente (Ej. CRM, Escaparate).
*   **[DOC] (Documentación):** Manuales técnicos, guías de estilo visuales o tutoriales de uso.
*   **[LOG] / [DEMO]:** Registros temporales, bitácoras de auditoría o experimentos aislados.
