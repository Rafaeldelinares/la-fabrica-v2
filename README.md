# 🏭 LA FÁBRICA IA

Integración de múltiples IDEs (Qwen + Antrigravity + Claude.ai) para desarrollo de aplicaciones web con PHP, Python, n8n y ActivePieces.

## 📋 Estructura
```
knowledge/
├── 00_INICIO.md (índice maestro)
├── 01_arquitectura/ (constitución, topología de datos)
├── 02_proyectos/ (CRM, Escaparate, Monitor Reputación)
├── 03_estandares/ (Navy Industrial, SQL, nomenclatura)
├── 04_metodos/ (flujos de trabajo, checklists)
├── 05_skills_y_mcp/ (configuraciones para IDEs)
└── 06_infraestructura/ (despliegue, Docker)
```

## 🎯 Propósito

Crear un flujo unificado donde:
- **Qwen (VS Code)** genera código React, SQL, n8n
- **Antrigravity (Claude)** orquesta workflows y accede a BD
- **Claude.ai** planifica arquitectura y decisiones complejas
